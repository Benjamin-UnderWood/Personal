普通函数——一路到底(飞机，高铁)
generator函数 ——中间能停(出租车)

--------------------------------------------------------------------------------

// 为什么让出租车停，拿东西，见朋友；
// 请求数据，不是瞬间回来的，是需要时间的，通过暂停等它结束，等数据过来

function *函数(){
    // 代码...
    $.ajax(xxx, function (){
        // 代码... 请求数据
    });
}

function *函数(){
    // 代码...

    yield $ajax(xxx); // 获取数据的相关操作

    // 代码...

}

// 普通函数涉及到一些异步操作那只能用回调的方式，等什么时候完了，再叫你回调
// generator 停着等着你，什么时候数据回来了，再接着走

--------------------------------------------------------------------------------
怎么做到走走停停的

生成两个小函数

function *show(){
    alert('a');

    yield; // 放弃；暂时放弃执行，把我的控制权交出去，过一会需要重新执行把控制权还回来

    alert("b");
}
// 原理，生成两个小函数
function show_1(){
    alert('a');
}
function show_2(){
    alert('b');
}
// 第一次next 走的是 a， 第二个next走的是 b
