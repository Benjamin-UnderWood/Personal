json的标准写法 // php java js 通用化的数据格式，因此有标准写法
1.只能用双引号
2.所有的名字都必须用引号包起来

{a: 12, b: 5}     // 错的
{"a": 12, "b": 5}   // 对的

{a: 'abc', b: 5}   // 错的
{"a": "abc", "b": 5}   // 对的
